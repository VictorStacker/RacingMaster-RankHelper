# 更新日志

本文档记录项目的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.0] - 2026-03-12

### 新增
- 🎯 **调教推荐功能**：在排位车辆推荐页面新增调教推荐列
  - 自动显示每辆车的调教参数
  - 支持多方案调教（如"调教1: 13232, 调教2: 23232"）
  - 支持带前缀调教（如"漂调教: 23332, 抓调教: 12332"）
  - 自动根据车辆阶位匹配对应调教数据（0阶匹配0阶调教，1-5阶匹配5阶调教）
  - 调教数据持久化存储，支持离线查看
- 🔄 **重置车辆数据库功能**：新增菜单项"数据(D) → 重置车辆数据库(R)"
  - 快速清空所有调教数据
  - 不影响账号数据
  - 重置后可重新抓取最新数据
- 🎨 **推荐计分车页面新增周次规则视觉标识**
  - 根据车辆在组别内的排名显示不同背景色
  - 第1-2周（前5名）：浅绿色背景
  - 第3-4周（前7名）：浅蓝色背景
  - 第5-7周（前9名）：浅灰色背景
  - 添加周次规则说明文字，帮助用户理解颜色含义

### 改进
- 🎨 **优化推荐计分车布局**
  - 移除"极限组"、"性能组"、"运动组"标签页中的"组别"列（冗余信息）
  - 移除"圈速总和"列（简化显示）
  - 调整列顺序为：排名、车型、阶数、圈速、调教推荐
  - 调教推荐列支持文本换行，自动调整行高
  - 移除了推荐页面的前3名高亮功能，统一使用周次规则背景色
  - 优化了表格视觉层次，更清晰地展示不同周次下的计分车辆范围
- 📊 **数据更新优化**
  - 一键同时更新圈速数据和调教数据
  - 更新进度显示更详细的统计信息

### 修复
- 🐛 修复调教数据解析问题
  - 修复无前缀多方案调教数据覆盖问题（如"13232 23232"）
  - 修复调教数据格式化问题（保持原始5位数字格式）
  - 修复特殊格式调教数据解析失败问题（如"漂 抓都是21232"）
  - 受影响车辆：科尼塞克 CCX、领克 03+TCR、宝马 M4 CSL等9款车辆

### 技术改进
- 📝 完善测试覆盖
  - 新增bug条件探索测试（10个测试用例）
  - 新增保持性属性测试（10个测试用例，包括基于属性的测试）
  - 所有测试通过，无回归问题
- 🏗️ 代码优化
  - 改进调教数据解析器逻辑
  - 增强数据库管理器功能
  - 优化缓存机制

### 说明
- ⚠️ 本程序仅针对《巅峰极速》国服（中国大陆服务器）


## [1.0.0] - 2026-03-11

### 新增
- 🎯 完整的联赛系统支持（14个联赛等级）
- 🚗 多账号管理功能
- 📊 车辆排行榜（支持按组别筛选）
- 🏆 计分车推荐系统（支持不同联赛等级的计分车辆配置）
- 💾 数据导入导出功能
- 🌐 更新车辆数据库功能
- 🖥️ 图形用户界面

### 功能特性
- 车辆排行榜
  - 实时查看所有车辆圈速数据
  - 按组别筛选（极限组/性能组/运动组）
  - 支持排序和搜索
  
- 我的车库
  - 多账号管理
  - 添加/删除车辆
  - 内联调整阶数
  - 自动按圈速排序
  
- 排位车辆推荐
  - 14个联赛等级配置
  - 精确的计分车辆数量
  - 分组别推荐
  - 全部推荐（综合排序）
  - 手动调整计分车辆数量


## 版本说明

### 版本号格式
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新类型
- **新增**：新功能
- **变更**：现有功能的变更
- **弃用**：即将移除的功能
- **移除**：已移除的功能
- **修复**：Bug 修复
- **安全**：安全相关的修复
