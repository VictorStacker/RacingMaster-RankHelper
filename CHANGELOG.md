# 更新日志

本文档记录项目的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-12-19

### 新增
- 🎯 完整的联赛系统支持（14个联赛等级）
- 🚗 多账号管理功能
- 📊 车辆排行榜（支持按组别筛选）
- 🏆 智能推荐系统（支持不同联赛等级的计分车辆配置）
- 💾 数据导入导出功能
- 🌐 自动从网站更新车辆数据
- 🖥️ PyQt6 图形用户界面
- 📝 完整的日志系统

### 功能特性
- 车辆排行榜
  - 实时查看所有车辆圈速数据
  - 按组别筛选（极限组/性能组/运动组）
  - 支持排序和搜索
  
- 我的车库
  - 多账号管理
  - 添加/删除车辆
  - 内联调整阶数
  - 自动按圈速排序
  
- 排位车辆推荐
  - 14个联赛等级配置
  - 精确的计分车辆数量
  - 分组别推荐
  - 全部推荐（综合排序）
  - 手动调整计分车辆数量

### 技术栈
- Python 3.10+
- PyQt6 (GUI)
- SQLAlchemy + SQLite (数据库)
- Playwright 
- pytest + Hypothesis (测试)

## 版本说明

### 版本号格式
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新类型
- **新增**：新功能
- **变更**：现有功能的变更
- **弃用**：即将移除的功能
- **移除**：已移除的功能
- **修复**：Bug 修复
- **安全**：安全相关的修复
