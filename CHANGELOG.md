# 更新日志

本文档记录项目的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循两段式格式（主版本号.次版本号）。

## [1.4] - 2026-03-15

### 修复
- 🐛 **修复车库车辆无法降阶问题**：调整阶数时改为直接更新车辆记录，避免先删后加导致的「此车已在车库中」错误，现在升阶和降阶均可正常操作
- 🖼️ **修复任务栏图标显示问题**：将 icon.ico 打包进 exe，并设置 Windows AppUserModelID，任务栏现在正确显示应用图标

## [1.3] - 2026-03-14

### 修复
- 🐛 **修复多账号数据导出问题**：导出功能现在会导出所有账号的车库和当前组合数据，而不仅仅是激活账号的数据
- 🐛 **修复切换账号时车库下拉框不同步问题**：从上方菜单切换账号后，车库页面的下拉框和 ✓ 标记现在会正确更新
- 🐛 **修复状态栏账号显示消失问题**：刷新视图后左下角当前账号名称不再消失

### 改进
- 📦 **重新设计导出/导入文件格式**：采用统一的多账号 JSON 格式，每个账号包含账号名称、描述、车库数据和当前组合数据
- 🔄 **导入时支持账号匹配**：根据账号名称匹配现有账号，不存在则自动创建新账号
- 🎛️ **导入时支持选择性恢复**：导入多账号数据时，提供账号选择对话框，用户可选择导入全部或特定账号
- 🧹 **简化导入/导出菜单**：移除冗余选项，统一为「导入车库数据」与「导出所有数据」两个入口
- 🏆 **车库新增总榜排名列**：我的车库第一列改为显示该车辆在全部车辆中的总榜排名（按圈速），取代原本的车辆 ID
- 🎯 **记忆联赛等级选择**：推荐页面的最高联赛等级选择现在会记住上次的设定，重启后不再重置为预设值
- 🏷️ **优化车库按钮文字**：将「删除」按钮改为「移出车库」，更清楚表达操作意图
- 🔢 **账号排序功能**：账号管理对话框新增「↑ 上移」「↓ 下移」按钮，可自由调整账号显示顺序
- 📊 **推荐页表头优化**：将「排名」改为「推荐序」，避免与车辆总榜排名混淆
- 🎨 **车库调整按钮居中**：操作列按钮现在正确居中显示
- 🪟 **简化数据更新进度对话框**：移除详细日志文字框，完成后显示结果并提供 5 秒倒计时自动关闭
- ℹ️ **改善关于对话框**：新增数据来源、GitHub 链接、免责声明及署名

### 说明
- ⚠️ 本程序仅针对《巅峰极速》国服（中国大陆服务器）


## [1.1] - 2026-03-12

### 新增
- 🎯 **调教推荐功能**：在排位车辆推荐页面新增调教推荐列
  - 自动显示每辆车的调教参数
  - 支持多方案调教（如"调教1: 13232, 调教2: 23232"）
  - 支持带前缀调教（如"漂调教: 23332, 抓调教: 12332"）
  - 自动根据车辆阶位匹配对应调教数据（0阶匹配0阶调教，1-5阶匹配5阶调教）
  - 调教数据持久化存储，支持离线查看
- 🔄 **重置车辆数据库功能**：新增菜单项"数据(D) → 重置车辆数据库(R)"
  - 快速清空所有调教数据
  - 不影响账号数据
  - 重置后可重新抓取最新数据
- 🎨 **推荐计分车页面新增周次规则视觉标识**
  - 根据车辆在组别内的排名显示不同背景色
  - 第1-2周（前5名）：浅绿色背景
  - 第3-4周（前7名）：浅蓝色背景
  - 第5-7周（前9名）：浅灰色背景
  - 添加周次规则说明文字，帮助用户理解颜色含义

### 改进
- 🎨 **优化推荐计分车布局**
  - 移除"极限组"、"性能组"、"运动组"标签页中的"组别"列（冗余信息）
  - 移除"圈速总和"列（简化显示）
  - 调整列顺序为：排名、车型、阶数、圈速、调教推荐
  - 调教推荐列支持文本换行，自动调整行高
  - 移除了推荐页面的前3名高亮功能，统一使用周次规则背景色
  - 优化了表格视觉层次，更清晰地展示不同周次下的计分车辆范围
- 📊 **数据更新优化**
  - 一键同时更新圈速数据和调教数据
  - 更新进度显示更详细的统计信息

### 修复
- 🐛 修复调教数据解析问题
  - 修复无前缀多方案调教数据覆盖问题（如"13232 23232"）
  - 修复调教数据格式化问题（保持原始5位数字格式）
  - 修复特殊格式调教数据解析失败问题（如"漂 抓都是21232"）
  - 受影响车辆：科尼塞克 CCX、领克 03+TCR、宝马 M4 CSL等9款车辆

### 技术改进
- 📝 完善测试覆盖
  - 新增bug条件探索测试（10个测试用例）
  - 新增保持性属性测试（10个测试用例，包括基于属性的测试）
  - 所有测试通过，无回归问题
- 🏗️ 代码优化
  - 改进调教数据解析器逻辑
  - 增强数据库管理器功能
  - 优化缓存机制

### 说明
- ⚠️ 本程序仅针对《巅峰极速》国服（中国大陆服务器）


## [1.0] - 2026-03-11

### 新增
- 🎯 完整的联赛系统支持（14个联赛等级）
- 🚗 多账号管理功能
- 📊 车辆排行榜（支持按组别筛选）
- 🏆 计分车推荐系统（支持不同联赛等级的计分车辆配置）
- 💾 数据导入导出功能
- 🌐 更新车辆数据库功能
- 🖥️ 图形用户界面

### 功能特性
- 车辆排行榜
  - 实时查看所有车辆圈速数据
  - 按组别筛选（极限组/性能组/运动组）
  - 支持排序和搜索
  
- 我的车库
  - 多账号管理
  - 添加/删除车辆
  - 内联调整阶数
  - 自动按圈速排序
  
- 排位车辆推荐
  - 14个联赛等级配置
  - 精确的计分车辆数量
  - 分组别推荐
  - 全部推荐（综合排序）
  - 手动调整计分车辆数量
