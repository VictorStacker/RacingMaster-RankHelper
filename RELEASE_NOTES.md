# RacingMaster-RankHelper v1.1.0 发布说明

## 📦 下载

- **可执行文件**: `RacingMaster-RankHelper-v1.1.0.exe`
- **文件大小**: 164.48 MB
- **发布日期**: 2026年3月12日

## ✨ 新功能

### 1. 推荐计分车布局改进
- 优化了排位车辆推荐界面的布局
- 改进了车辆展示的视觉效果

### 2. 调教推荐功能
- 新增车辆调教数据显示
- 支持多种调教方案（漂移调教、抓地调教等）
- 自动从数据源获取最新调教数据

### 3. 重置车辆数据库
- 新增"数据(D) → 重置车辆数据库(R)"菜单项
- 可清空调教数据但保留账号数据
- 重置前显示确认对话框和数据统计

## 🐛 Bug 修复

### 调教数据解析修复
- 修复了多方案调教数据解析错误（如 "13232 23232"）
- 修复了带前缀调教方案的解析（如 "漂调教"、"抓调教"）
- 修复了特殊格式"都是"关键字的处理
- 移除了不必要的连字符格式化，保持原始5位数格式

**受影响车辆**（共9辆）:
- 科尼塞克 CCX
- 领克 03+TCR
- 宝马 M4 CSL
- 等其他车辆

## 🔧 技术改进

### 数据库路径优化
- 统一数据库存储位置到用户目录 `~/.racingmaster-rankhelper/`
- 主数据库: `database.db`
- 调教数据库: `tuning_data.db`
- 日志目录: `logs/`

### 打包优化
- 添加自定义应用图标
- 文件名包含版本号
- 不包含任何个人数据或测试数据

## 📋 适用范围

**本程序仅适用于巅峰极速国服（中国服务器）**

## 🚀 使用说明

### 首次使用
1. 下载并运行 `RacingMaster-RankHelper-v1.1.0.exe`
2. 点击"数据(D) → 更新数据库(U)"获取最新车辆和调教数据
3. 在"我的车库"标签页添加你的车辆
4. 在"排位车辆推荐"标签页查看推荐

### 数据更新
- 定期点击"数据(D) → 更新数据库(U)"获取最新数据
- 数据来源: [阿龙WayLong](https://waylongrank.top/index.html)

### 账号管理
- 支持多账号管理
- 点击"账号(A) → 管理账号(M)"进行账号切换
