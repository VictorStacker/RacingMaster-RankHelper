# RacingMaster-RankHelper v1.3 发布说明

## 📦 下载

- **可执行文件**: `RacingMaster-RankHelper-v1.3.exe`
- **发布日期**: 2026年3月14日

## ✨ 主要更新

### 🐛 修复
- **修复多账号数据导出问题**：导出功能现在会导出所有账号的车库和当前组合数据，而不仅仅是激活账号的数据
- **修复切换账号时车库下拉框不同步问题**：从上方菜单切换账号后，车库页面的下拉框和 ✓ 标记现在会正确更新

### 🎯 改进
- **车库新增总榜排名列**：我的车库第一列改为显示该车辆在全部车辆中的总榜排名（按圈速），取代原本的车辆 ID
- **记忆联赛等级选择**：推荐页面的最高联赛等级选择现在会记住上次的设定，重启后不再重置为预设值
- **优化车库按钮文字**：将"删除"按钮改为"移出车库"，更清楚表达操作意图
- **重新设计导出/导入文件格式**：采用统一的多账号 JSON 格式
- **导入时支持账号匹配与选择性恢复**：根据账号名称匹配，并提供账号选择对话框
- **简化导入/导出菜单**：移除不必要的选项，操作更直觉

## 📥 安装方式

### 首次使用
1. 下载并运行 `RacingMaster-RankHelper-v1.3.exe`
2. 点击"数据(D) → 更新数据库(U)"获取最新车辆和调教数据
3. 在"我的车库"标签页添加你的车辆

### 从旧版本升级
直接替换旧版 exe 即可，数据库会自动保留。

## ⚠️ 适用范围

本程序仅针对《巅峰极速》国服（中国大陆服务器）。
