# RacingMaster-RankHelper v1.3 发布说明

## 📦 下载

- **可执行文件**: `RacingMaster-RankHelper-v1.3.exe`
- **发布日期**: 2026年3月14日

## ✨ 主要更新

### 🐛 修复
- **修复多账号数据导出问题**：导出功能现在会导出所有账号的车库和当前组合数据
- **修复切换账号时车库下拉框不同步问题**：从菜单切换账号后，下拉框和 ✓ 标记正确更新
- **修复状态栏账号显示消失问题**：刷新后左下角当前账号名称不再消失

### 🎯 改进
- **账号排序功能**：账号管理新增上移/下移按钮，可自由调整账号顺序
- **车库新增总榜排名列**：第一列改为显示总榜排名，取代原本的车辆 ID
- **记忆联赛等级选择**：推荐页面的联赛等级设定重启后不再重置
- **优化车库按钮文字**：「删除」改为「移出车库」
- **推荐页表头优化**：「排名」改为「推荐序」，避免与总榜排名混淆
- **简化数据更新进度对话框**：完成后显示结果，5 秒倒计时自动关闭
- **简化导入/导出菜单**：统一为「导入车库数据」与「导出所有数据」
- **改善关于对话框**：新增数据来源、GitHub 链接及署名

## 📥 安装方式

### 首次使用
1. 下载并运行 `RacingMaster-RankHelper-v1.3.exe`
2. 点击「数据(D) → 更新数据库(U)」获取最新车辆和调教数据
3. 在「我的车库」标签页添加你的车辆

### 从旧版本升级
直接替换旧版 exe 即可，数据库会自动保留。

## ⚠️ 适用范围

本程序仅针对《巅峰极速》国服（中国大陆服务器）。
