# RacingMaster-RankHelper v1.5 发布说明

## 📦 下载

- **可执行文件**: `RacingMaster-RankHelper-v1.5.exe`
- **发布日期**: 2026年3月21日

## ✨ 主要更新

### 🏷️ 车库休息标记
- 在「我的车库」中可直接将车辆标记为“已跑够/休息中”
- 标记按账号分别保存，不会互相干扰

### 🎯 推荐页显示优化
- 被标记休息的车辆仍会显示在推荐页中
- 这些车辆会被稳定地下沉到列表底部，方便你优先看还没跑够的车
- 车辆原本的**推荐序**会保留，不会因为置底显示而被改写

### 💾 数据兼容
- 导出所有数据时会一并保存车辆休息状态
- 重新导入后可恢复这些标记
- 旧数据库会自动补齐所需字段，无需手动重建数据库

## 📥 安装方式

### 首次使用
1. 下载并运行 `RacingMaster-RankHelper-v1.5.exe`
2. 点击「数据(D) → 更新数据库(U)」获取最新车辆和调教数据
3. 在「我的车库」标签页添加你的车辆
4. 对已跑够的车辆使用「标记休息」，然后回到推荐页生成推荐

### 从旧版本升级
直接替换旧版 exe 即可，本地数据库会自动保留，并在启动时完成必要迁移。

## ⚠️ 适用范围

本程序仅针对《巅峰极速》国服（中国大陆服务器）。
